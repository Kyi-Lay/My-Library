1. Initialization: Create a new local Git repository: git init Clone an existing remote repository: git clone 2. Staging: Add changes to the staging area: git add or git add . Check the staging area: git status 3. Committing: Commit changes with a descriptive message: git commit -m "Your commit message" 4. Pushing: Push changes to a remote repository: git push origin main (or your branch name) 5. Pulling: Fetch and merge changes from a remote repository: git pull origin main 6. Branching: Create a new branch: git branch Switch to a different branch: git checkout Merge branches: git merge 7. Other common commands: View commit history: git log Compare changes: git diff Undo changes: git reset HEAD or git revert
